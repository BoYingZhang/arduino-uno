# Arduino 伺服馬達控制專案

這個專案展示了如何使用 Arduino UNO 控制伺服馬達的基本操作。

## 硬體需求

- Arduino UNO 開發板
- SG90 伺服馬達
- 連接線

## 接線方式

1. 伺服馬達橙色線（訊號線）-> Arduino 數位腳位 9
2. 伺服馬達紅色線（VCC）-> Arduino 5V
3. 伺服馬達棕色線（GND）-> Arduino GND

## 功能說明

本程式實現了以下功能：
- 控制伺服馬達從 0 度到 180 度循環轉動
- 每次移動間隔 1 秒
- 透過序列埠監控當前角度

## 使用方法

1. 將程式碼上傳至 Arduino UNO 開發板
2. 開啟序列監控視窗（波特率：9600）
3. 觀察伺服馬達轉動情況及序列埠輸出

## 程式說明

主要程式位於 `src/servo_control.ino`，使用了 Arduino 內建的 Servo 函式庫來控制伺服馬達。
